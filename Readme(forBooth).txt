VRChatイベントカレンダー入力支援ツール
=======================================

VRChatイベントカレンダー (<https://vrceve.com/>) のイベント登録用フォームに、
イベント情報を自動入力する Windows 向けのツールです。

イベント運営のたびに同じ内容を何度も入力する手間を減らすことを目的に作成しました。
また、告知文などの定型文を作成・保存しておける入力フォームもおまけとして付属しています。

※本ツールは VRChatイベントカレンダー運営様とは一切関係のない、有志による非公式ツールです。

--------------------------------------------------
■ 同梱物（例）
--------------------------------------------------
- VRCカレンダー入力.exe   … ツール本体（ダブルクリックで起動）
- config.json               … 入力内容を保存する設定ファイル（自動作成）
- profile フォルダ         … Google ログイン情報などを保存するフォルダ（自動作成）
- cft フォルダ　　　　　　… 自動入力用ブラウザ(自動作成)
--------------------------------------------------
■ 動作環境
--------------------------------------------------
- Windows 10 以降
- インターネット接続環境
- Google アカウント

--------------------------------------------------
■ かんたんセットアップ手順
--------------------------------------------------
1. ダウンロードした ZIP を任意のフォルダに解凍します。
2. 解凍したフォルダ内の「VRCカレンダー入力.exe」をダブルクリックして起動します。
3. 画面上部の案内に沿って、次の順番で操作してください。

  (1) 「入力内容」タブ
      - イベント名、日時、主催者、イベント内容などを入力します。

  (2) 「実行 / ログ」タブ
      - 初回のみ「プロファイル作成」ボタンを押し、表示されるブラウザで
        Google アカウントにログインしてください。
      - ログイン完了後、ブラウザを閉じるとプロファイルが保存されます。
      - その後「フォーム自動入力」ボタンを押すと、Google フォームが開き、
        設定内容を自動で入力します。(投稿までは行いません)

  (3) 「おまけ」タブ
      - イベントの告知などで使うテンプレート文を作成・保存できます。
        （メイン機能とは無関係です。必要な方のみご利用ください）

--------------------------------------------------
■ 注意事項
--------------------------------------------------
- 初回実行時に、自動入力用ブラウザとして Google 公式の Chrome for Testing をツールと同じフォルダ内に自動ダウンロードします（数百MB程度）
- Google アカウントのログイン情報は、解凍したフォルダ配下の profile フォルダに保存されます。
  他人とフォルダを共有する際は、この profile フォルダを含めて渡さないよう十分ご注意ください。
- フォームの仕様変更やサービス側の都合により、将来動作しなくなる可能性があります。
- VRChat および VRChatイベントカレンダーの利用規約・ガイドラインに反しない範囲でご利用ください。
